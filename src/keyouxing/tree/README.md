# tree(2019.4.1)
## TreeNode(2019.4.1)
1. 说明
    * 树的节点
    
## Tree(2019.4.1)
1. 说明
    * 二叉树的封装类，根据前序和中序构造树（int[] pre, int[] in）

## PrintTree(2019.4.1)
1. 题目
    * 打印二叉树（层次遍历一行一行打印节点）
2. 说明
    * 重新定义了树节点的结构，增加(position)字段，表示当前节点在所在行的位置
    
## GetNext(2019.4.2)
1. 题目
    * 二叉树的下一个节点
2. 说明
    * 给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。
